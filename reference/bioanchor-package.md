# bioanchor: Permanent Archiving of MCMC Analysis Metadata on Arweave

Bridges Bayesian MCMC analysis in R with Arweave decentralized storage
for permanent, tamper-proof archiving of analysis metadata. Extracts
convergence diagnostics, posterior summaries, and model specifications
from 'Stan', 'brms', and 'coda' objects, then uploads them as structured
JSON to the Arweave permaweb.

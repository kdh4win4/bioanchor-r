## Package description

bioanchor bridges Bayesian MCMC analysis in R with Arweave 
decentralized storage for permanent archiving of analysis metadata.
This is a companion R implementation to the Python BioAnchor package 
on PyPI (https://pypi.org/project/bioanchor/).
